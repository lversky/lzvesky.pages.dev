# Personal Link in Bio

Welcome to my personal Link in Bio project! This repository contains the source code for a simple, visually appealing webpage that consolidates all of my important links in one place.

## Overview
This project is designed to provide a single hub where visitors can quickly access my social media profiles, portfolio, and other key online destinations. It is mobile-friendly and optimized for an easy, clean browsing experience.

## Features
- Clean, responsive design
- Centralized collection of my personal links
- Easy to navigate and laptop-first layout (can be used on Mobile).

## Live Site
You can view the live version of my Link in Bio page here: [LzveSky.pages.dev](https://lzvesky.pages.dev/)
